# Descrição
Shell Script de pós instalação do Linux Mint 19.x para utilização pessoal.

### Alguns dos pacotes instalados:
- Google Chrome
- Insync
- Wine
- Simple Note
- 4k Video Downloader
- Virtualbox
- snap
- spotify
- slack
- skype
